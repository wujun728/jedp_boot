Introduce

SpringSide是以Spring Framework为核心的，Pragmatic风格的JavaEE应用参考示例，是JavaEE世界中的主流技术选型，最佳实践的总结与演示。

  • Quickstart: 一个迷你的TodoList应用。
  • Showcase: 五花八门的JavaEE技术大派对。

How to contribute

Github提倡社交网络式的开源生活，无需申请开发团队成员资格，人人都可以为SpringSide贡献代码。Just "Fork" and send the "Pull Request"，请看How to Contribute。
